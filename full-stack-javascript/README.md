# Full Stack JavaScript

Exercises & the curriculum project completed as part of [The Odin Project&#39;s Full Stack JavaScript](https://www.theodinproject.com/paths/full-stack-javascript) course, the developer path chosen over [Full Stack Ruby on Rails](https://www.theodinproject.com/paths/full-stack-ruby-on-rails).

As with the rest of this repository, everything here is written without AI assistance - solutions come from the official curriculum, MDN, Stack Overflow, and the [The Odin Project Discord server](<https://www.theodinproject.com/paths#:~:text=All%20Paths-,Community,-Support%20us>). AI is used only as an absolute last resort, and only for hinting guidance, never as a result machine.

## Directories

./* - Each subcourse with its own lessons, exercises and projects have been put into each subfolder to keep track of it more easily.

*/exercises - Smaller exercises and practice problems completed throughout the course, organized by lesson/topic.

*/curriculum-projects - The larger project(s) required to complete the course.
